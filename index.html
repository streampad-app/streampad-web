FAQ

よくある質問

OBS Studioとの接続方法は？
  1. OBS Studio 28以上をインストール
  2. OBS → ツール → WebSocketサーバー設定 → 「WebSocketサーバーを有効にする」をON
  3. ポート番号を確認（デフォルト: 4455）、必要ならパスワードを設定
  4. StreamPad Companion → メニューバー → 設定 → OBS Studio → URLに ws://localhost:4455 を入力
  5. パスワードを設定している場合は入力
  6. 「保存＆テスト」をクリックして接続確認
StreamPadは無料で使えますか？

はい！基本機能は全て無料でお使いいただけます。Proにアップグレードすると、10バンドEQ、Magic Boost、Output Groups、サンプルレート設定などの高度な機能が解放されます。

Windows対応していますか？

はい！Windows Companion（.NET 8）を無料でダウンロードできます。Windows 10 2004以降 / Windows 11に対応しています。

MIDIコントローラーとして使うには？

Companionを起動するだけで仮想MIDIデバイス「StreamPad」が自動作成されます。ダイヤル・フェーダー・ボタンの編集画面でアクションタイプを「MIDI」に変更し、CC番号やノート番号を設定してください。GarageBand、Logic Pro、Ableton Live等のDAWで自動認識されます。

FAQ

Frequently Asked Questions

How do I connect to OBS Studio?
  1. Install OBS Studio 28 or later
  2. OBS → Tools → WebSocket Server Settings → Enable "Enable WebSocket server"
  3. Note the port number (default: 4455), set a password if needed
  4. StreamPad Companion → Menu Bar → Settings → OBS Studio → Enter ws://localhost:4455 as URL
  5. Enter the password if you set one
  6. Click "Save & Test" to verify the connection
Is StreamPad free to use?

Yes! All core features are completely free. Upgrade to Pro to unlock advanced features like 10-band EQ, Magic Boost, Output Groups, sample rate settings, and more.

Does it support Windows?

Yes! The Windows Companion (.NET 8) is available as a free download. It supports Windows 10 2004+ and Windows 11.

How do I use StreamPad as a MIDI controller?

Just launch Companion and a virtual MIDI device "StreamPad" is automatically created. In the edit screen for dials, faders, or buttons, change the action type to "MIDI" and configure the CC number or note number. It is automatically recognized by DAWs like GarageBand, Logic Pro, and Ableton Live.
